⚙️ Cómo funciona este tracker (versión pastel)

01

Captura de IP

Leemos HTTP_X_FORWARDED_FOR o REMOTE_ADDR para obtener la IP real del visitante.

02

Geolocalización

Se consulta ip-api.com (o servicio similar) para obtener país/ciudad aproximados y ISP.

03

Persistencia

Los datos se guardan en la base de datos definida (SQLite para desarrollo, PostgreSQL recomendado en producción).
